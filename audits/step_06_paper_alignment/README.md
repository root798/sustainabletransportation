# step_06_paper_alignment

Paper-facing support files for manuscript and response-letter finalisation. Do NOT edit the manuscript or rebuttal source directly; apply the change maps listed below.

## Files

| file | purpose |
| --- | --- |
| `RESULTS_ALIGNMENT.md` | Paste-ready Results paragraphs (R1–R8). |
| `METHODS_ALIGNMENT.md` | Paste-ready Methods paragraphs (M1–M8). |
| `FIGURE_INSERTION_MAP.md` | Mapping from manuscript figure slots to files in `reports/paper_support/figures/`. |
| `CAPTION_ALIGNMENT.md` | Publication-safe captions (machine-generated, do not re-word numeric claims). |
| `TABLE_SANITIZATION.md` | Table-by-table instructions (T1–T7). |
| `REVIEWER_RESPONSE_FINAL.md` | Full response-letter paragraph + short-answer version. |
| `AUTO_REVIEW_REPORT.md` | Repo-wide grep for forbidden wording hazards; every hit is advisory. |
| `MANUSCRIPT_CHANGE_MAP.md` | 11-item edit checklist for the external manuscript. |
| `REBUTTAL_CHANGE_MAP.md` | 8-item edit checklist for the external response letter. |
| `STAGE_1_COMPLETION_REPORT.md` | Stage completion summary + validation cross-reference. |

## Scope

CA / OH only. U.S. Average is quarantined for paper-facing quantitative comparison.
